# IMC_Grafica_circular
Calculadora donde  se calcula el imc de las personas y se gusrada en una tabla donde se pueden visualiar en una grafica lineal y una circular 

* para obtener las dependencias deves  ejecutar npm install !!!!!
* si tienes alguna dependencia de reac desactualizada ejecuta este comando " npx expo install --fix " 
* deves istalar la sigiente dependencia para la grafica circular "npm install react-native-chart-kit"!!!

* * tambien "npm install react-native-svg"
* Por si alguna depenedecia causa problemas al instalarla "npx react-native start --reset-cache"
